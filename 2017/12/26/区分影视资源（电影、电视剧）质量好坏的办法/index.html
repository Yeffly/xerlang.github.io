区分影视资源（电影、电视剧）质量好坏的办法

发表于 2017-12-26 | 分类于 02.影视

前言

在国内，因为种种原因，大家看电影、电视剧等影视作品（特别是国外的作品）的时候，都喜欢下载下来看，一是因为很久很久之前，网速不行，在线看视频卡，到现在养成的习惯没法改，二是有些资源不在视频网站上上架或者上架很晚（国内美剧审查），所以……

提到下载，大家最关注的就是清晰度，网上一个资源的盗版有的是，好的会写清晰度片源啦，懒得就直接写“高清”、“超清”等等，导致枪版都敢自称为“超清”，不忍直视。想像一下，名字写的高清无码，下下来才发现全是马赛克是什么感觉。

所以，写点东西来讨论一下怎么区分资源质量的好坏。

一、根据资源名称区分

资源名称是个好东西，但是很多资源发布者为了方面，直接就写个超清，根本无法分辨这个资源质量的好坏，但是稍微正规的网站、字幕组发布的资源名称都不会这么简单，其中包含着很多有用的信息。

以我收藏的一个资源为例。

忽略这些歪歪扭扭的数。一般稍正规的字幕组发布的资源都会包含这些信息，只是顺序可能不同，没什么影响。

资源名就不说了，只说有用的。

  1. 年份：影片上映的年份。
  2. 版本：导演剪辑版啦、加长版啦等等。
  3. 分辨率：比较常见的就是1080p和720p。
  4. 片源：这个要说的比较多，片源包括BluRay（蓝光）、WEB-DL、HDTV、TS等等。详细的后面说。
  5. 视频编码
  6. 音频编码
  7. 音轨

一般资源名称就包括这些内容，更为详细的信息会包括在随资源一起的nfo文件中。

其实从解释上来看，3和4决定着资源质量的好坏，所以详细来说说这个。

片源包括很多种：BluRay、HDDVDRip、WEB-DL、HDRip(HDTV)、HR-HDTV、DVDRip、TC、TS、CAM等等，这里说几个常见的，其他的可以自行百度。

  1. BluRay：蓝光，这是目前资源质量最好的片源，来源就是厂商发布的蓝光碟。其中又可以分为原盘、转制等等，也是收藏的首选。
  2. WEB-DL：顾名思义，从网页上下载的资源，来源是各大正规视频网站播放后转制的。相比于蓝光，清晰度略逊一筹，可是资源大小却很有优势。而且WEB-DL出现的时间也会早蓝光一到两个月，是尝鲜的首选。
  3. HDTV：电视片源转制的，很早之前网络不发达，电影上映之后会放到电视上播，转录后就是HDTV，现在电影一般没有HDTV了，但是电视剧很多的，好的HDTV资源的清晰度比WEB-DL差不了多少。
  4. TS：枪版，来源是影院盗录，画质相当不好，发黑、晃动。

所以，从以上来看，大家下载资源的时候，最好是下载信息齐全的资源，质量有保证，所以，抛弃所谓的“超清”、“高清”吧，写着“超清”、“高清”的可能是枪版，而且质量（清晰度、字幕质量等等）肯定没有正规网站或字幕组发布的好。

二、根据资源大小区分

但是，很多情况下，同时蓝光版，但是资源大小区别很大，从几G到几十G不等，这又是为什么呢？

因为蓝光版又分蓝光原盘、转制版等等，大小不一定，但是也有一定的规律，可以进行参考。

以2小时的资源为例：

  • 原盘大小在30G以上。
  • 转制1080p蓝光在10G以上，再压制会在4-8G之间。
  • 转制720p蓝光在5G以上，再压制会在3-5G之间。
  • WEB-DL在2-3G之间。
  • 枪版大小不定。

如果一个资源号称蓝光，但只有1G左右，连下都不用下，假的。

三、根据发布时间区分

资源根据片源的不同，发布时间会有不同，根据时间，可以加以区分。

  1. 蓝光：好莱坞电影，在影院下映后一般三个月左右会发售蓝光碟；国内电影发不发售蓝光碟都是个问题。
  2. WEB-DL：好莱坞电影，影院下映后一个月左右会上架国外视频网站，上架国内视频网站的时间不一定；国内电影影院下映后一到两个月会上架国内视频网站。

所以，好莱坞电影，影院下映之后一个月左右出WEB-DL资源，三个月左右出蓝光资源。资源出来后，字幕组翻译压制快得两三天慢的十来天就行了。国内电影，下映后一到两个月会出WEB-DL资源，蓝光时间不定。

所以，在这些时间之前出来的资源，不是泄漏的话，那绝对是枪版，没别的。（这里的泄漏是个宽泛的概念，之后详细解释）。所以，今天一个帖子里说的现在《大圣归来》有高清、超清了，我不用下载就敢说，那些都是枪版，绝无例外。

四、根据字幕组选择

根据以上三个办法，不必下载，就可以大致判断一个资源质量的好坏了。当然，对国外电影来说，还有一种办法，那就是根据字幕组或资源组，优秀的字幕组或资源组发布的资源有品质保证。

附录

（一）资源泄露

经过上边讲的，高质量资源的发布时间一般是一定的，但是有些特殊情况下，资源在影片还未上映或上映之初就已经流出，并且明显不是枪版，那就涉及到资源泄漏。

资源泄露一般包括以下几种情况：

  1. 厂商被黑，资源泄露。比如《敢死队3》、《金刚狼2》，因为厂商被黑，导致资源在上映之初流出。
  2. 各大市场上映时间错开，导致资源流出。比如一部电影在内地上映的时间比北美上映晚两个月，那么影片北美上映之后一个月左右上架视频网站，资源就会流到内地，那么内地观众在影片上映之前就会看到高质量资源。虽然一般发行方都会尽力避免这种情况出现，但是也有，比如最近的韩版资源，在内地上映之前，韩版资源就会流出。比如《小黄人大眼萌》，内地上映之前韩版资源早已流出。
  3. 影院泄露。这里指的不是盗录的枪版，而是影院内外勾结拿到高质量资源。这种情况以前我没听说过，今年第一次听说，就是《港囧》。

总结

以上是我这么多年寻找资源的一些心得，下载盗版资源有的时候是不得已，因为有些资源国内根本没有正版。不过还是希望大家多多支持正版。我的原则是喜欢的，能上映的上映时去影院支持，有资源了下载下来收藏。

收藏一般是国外电影收藏蓝光，国内有蓝光收藏蓝光，没有就WEB-DL。一概不看枪版，对眼对影片都是一种不尊重。
